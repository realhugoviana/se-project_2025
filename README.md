# Bit packing for integer compression

### Requirements

openjdk 21
maven

### How to use

1) execute this : mvn exec:java -Dexec.mainClass="com.hugoviana.App"
2) you will be asked to enter the path of a csv file or to enter manually an array.
3) Then, you have to chose the compression type between the options given
4) Then you have the option of either look up an integer by index or to decompress the array by tiping -1
5) ctrl+c to exit
