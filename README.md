# Bit packing for integer compression

### Requirements

openjdk 21
maven

### How to use

1) execute this : mvn exec:java -Dexec.mainClass="com.hugoviana.App"
2) Enter the path of a csv file or enter manually an array.
3) Chose the compression type between the options given
4) Either look up an integer by index or decompress the array by tiping -1
5) ctrl+c to exit
